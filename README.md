# Personal Profile Website

A modern personal profile webpage built with HTML5 and CSS3.  
This project features a clean layout, responsive design, styled navigation bar, and custom color themes using CSS variables.

- Semantic HTML structure
- External CSS styling
- Responsive layout
- Navigation hover effects
- Google Fonts integration
